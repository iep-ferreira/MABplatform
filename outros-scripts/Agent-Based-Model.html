Funo que cria o ambiente (paisagem)

O usurio pode configurar o tamanho da rede l.size, o vetor nmero de fragmentos n.patches, com trs nveis de recurso (pobre, intermedirio, rico), e cobertura de cada tipo de fragmento cover (soma da cover < 1).

  • default para l.size 100 px,
  • default para n.patches 2 fragmentos de cada tipo,
  • default para cover de 25% para cada tipo de fragmento.
# Criando a Paisagem
land <- land.create(l.size = 200, n.patches =  c(1, 2, 3), cover = c(0.20,0.15,0.30))

Movimentao dos animais

  • As simulaes representam 21 semanas consecutivas, em que cada passo de tempo dura uma hora.

  • O movimento uma adaptao do caminho aleatrio, mas considerando tempos de permanncia proporcionais qualidade do recurso.

  • As 20 semanas inciais so para a fase de adaptao. Assim os indivduos podem estabelecer o padro natural de ocupao a partir de posies iniciais.

  • A simulao da deposio de vestgios e coleta dos dados ocorre nas duas semanas finais.

  • Nas simulaes, a taxa individual de produo de vestgios foi fixada em quatro vestgios por dia.

    • n o tamanho populacional
    • ra o argumento que recebe o raster da paisagem
    • time o nmero de semanas (adaptao + coleta de dados)
    • where onde posicionamos os animais no incio da populao. (all para posicion-los em todo o ambiente e resources para apenas aonde tem recursos).
ra <- land$raster
mov <- animal.mov(n = 20, ra = ra , time = 21, where = "resource")

Para visualizar a movimentao animal

  • Com adapt = FALSE visualizamos apenas as duas semanas finais;

  • Com adapt = TRUE visualizamos todas as semanas, incluindo a fase de adaptao.

tracks.viewer(obj = mov, r = ra, adapt = FALSE)

Para fixar os transectos

  • Os transectos podem ser linhas retas ou caminhos aleatrios (h um parmetro de convoluo);

  • Eles podem ser fixados em toda a rea, aleatoriamente, ou apenas aonde tem recurso:

    • sampling == random significa amostragem aleatria;

    • sampling == random amostragem por convenincia.
